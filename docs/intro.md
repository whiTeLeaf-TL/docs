# 👋 服务器介绍

> 欢迎加入 **TL | 白叶城** 多玩法 Java 服务器 💙

白叶城服务器的全名为 **whiTe-Leaf 白叶**，这是一个致力于为玩家提供多样化游戏体验的新兴社区。

我们提供以下主要玩法：

- **建筑**：展示你的创造力，设计并建造令人惊叹的建筑作品。
- **生存**：在原版生存模式中挑战自我，与环境斗智斗勇。
- **红石**：探索和使用红石机制，打造自动化装置和复杂的电路系统。
